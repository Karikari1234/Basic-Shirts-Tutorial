Planning Gantt Chart

Planning Gantt Chart

The Planning Gantt Chart provides a visual timeline of all production activities. You can view scheduled production for each order, track progress, and identify potential bottlenecks.

Planning Gantt Chart সমস্ত production activities-এর একটি visual timeline প্রদান করে। আপনি প্রতিটি order-এর জন্য scheduled production দেখতে পারেন, progress track করতে পারেন এবং সম্ভাব্য bottlenecks চিহ্নিত করতে পারেন।

  • View all scheduled production on a timeline
  • Color-coded status indicators for quick reference
  • Filter view by floor, order, or date range
  • Timeline-এ সমস্ত scheduled production দেখুন
  • দ্রুত reference-এর জন্য color-coded status indicators
  • Floor, order, বা date range দ্বারা view filter করুন
URL: http://139.162.34.211:8080/admin_panel/planning

Replan Feature

Replan Feature

The Replan feature allows planners to adjust production schedules when changes occur. After modifying the plan, remember to click the "Publish" button to make your changes visible.

Replan feature planners-দের পরিবর্তন ঘটলে production schedules সামঞ্জস্য করতে দেয়। Plan পরিবর্তন করার পরে, আপনার পরিবর্তনগুলি দৃশ্যমান করতে "Publish" বোতাম ক্লিক করতে ভুলবেন না।

  • Modify existing production schedules
  • Adjust timelines for delays or expedited orders
  • Reallocate resources between production lines
  • Changes are automatically reflected in the Hourly Production Sheet
  • বিদ্যমান production schedules পরিবর্তন করুন
  • Delays বা expedited orders-এর জন্য timelines সামঞ্জস্য করুন
  • Production lines-এর মধ্যে resources পুনরায় বরাদ্দ করুন
  • পরিবর্তনগুলি স্বয়ংক্রিয়ভাবে Hourly Production Sheet-এ প্রতিফলিত হয়
URL: http://139.162.34.211:8080/admin_panel/planning/replan
